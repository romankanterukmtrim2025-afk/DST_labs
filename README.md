## Лабораторна робота 6 - Message Queues

## Запуск
Аргументи: lv.rst.uk.to 61616

## Команди
- ping
- echo <текст>
- login <логін> <пароль>
- list
- msg <користувач> <текст>
- file <користувач> <шлях_до_файлу>
- exit

## Примітка
При запуску без діючого сервера виводиться:
"Connection failed: Could not connect to broker URL"

## Бонус
Якщо користувач не вводив команди більше 5 хвилин,
програма автоматично відповідає на вхідні повідомлення:
"Sorry, I'm AFK, will answer ASAP"
